Tile Stitching and TIFF Saving Script Documentation

Overview

This script stitches tiles from a specified directory into a single array and saves the result as a TIFF file. It involves loading tile data, stitching them together, and saving the final result. The script also logs memory usage at various stages of the process.

Functions

print_memory_usage()

Logs the current memory usage of the process.

stitch_tiles_from_directory(input_dir, original_shape, tile_shape)

Stitches tiles from a directory into a single array.

Parameters:

  • input_dir (str): Directory containing input tile files.
  • original_shape (tuple): Original shape of the full dataset.
  • tile_shape (tuple): Shape of each tile.

Returns:

  • np.ndarray: Stitched result array.

main(input_dir, output_tiff_path, original_shape, tile_shape)

Main function to stitch tiles and save the result as a TIFF file.

Parameters:

  • input_dir (str): Directory containing input tile files.
  • output_tiff_path (str): Path to save the output TIFF file.
  • original_shape (tuple): Original shape of the full dataset.
  • tile_shape (tuple): Shape of each tile.
